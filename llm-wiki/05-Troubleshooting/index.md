# 故障排查索引

> 常见问题排查指南

## 排查分类

### 训练问题

- GPU 内存不足 — 调整 batch size 或使用 ZeRO
- 训练收敛慢 — 检查学习率和数据质量
- 分布式训练失败 — 检查网络配置和 NCCL

### 推理问题

- 推理速度慢 — 启用 KV Cache 或量化
- 模型加载失败 — 检查模型格式和依赖
- 显存溢出 — 使用模型并行或量化

### 环境问题

- CUDA 版本不匹配 — 检查驱动和 CUDA 版本
- 依赖冲突 — 使用虚拟环境隔离
- 权限问题 — 检查文件权限和用户组

## 排查工具

| 工具 | 用途 |
|------|------|
| nvidia-smi | GPU 状态监控 |
| htop | 系统资源监控 |
| dmesg | 内核日志查看 |
| journalctl | 系统日志查看 |

## 排查流程

1. **收集信息** — 错误日志、系统状态
2. **定位问题** — 分析错误原因
3. **尝试修复** — 应用解决方案
4. **验证结果** — 确认问题解决
