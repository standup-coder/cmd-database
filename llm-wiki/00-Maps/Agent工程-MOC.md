# AI基础设施/Agent工程

> AI Agent框架与工程化工具，涵盖LLM应用编排、多Agent协作、RAG、提示工程

## 命令列表

- 🟡 [[agno]] — Agno (原Phidata) 轻量级Agent框架，支持记忆、知识库、工具、多模态
- 🔴 [[autogen]] — Microsoft AutoGen多Agent对话框架，支持代码生成、工具调用、人机协同
- 🟡 [[crewai]] — CrewAI多Agent协作框架，角色扮演、任务委派、团队协作执行复杂工作流
- 🟡 [[dify]] — Dify开源LLM应用开发平台，可视化编排Agent、RAG、工作流，支持自托管
- 🟡 [[dspy]] — DSPy声明式LLM编程框架，优化提示和权重而非手工工程
- 🟡 [[embedchain]] — EmbedChain RAG框架，简化数据加载、分块、嵌入、查询全流程
- 🟡 [[flowise]] — Flowise可视化LangChain工作流构建器，拖拽式创建LLM应用
- 🟡 [[guidance]] — Guidance可控生成库，通过语法约束LLM输出结构，确保JSON/代码格式正确
- 🟡 [[haystack]] — Haystack端到端NLP框架，支持RAG、Agent、文档搜索、Pipeline编排
- 🟡 [[instructor]] — Instructor结构化输出库，基于Pydantic验证LLM响应，确保类型安全
- 🟡 [[langchain]] — LangChain LLM应用开发框架，支持Chains、Agents、RAG、工具调用
- 🟡 [[langgraph]] — LangGraph状态机Agent框架，支持循环、条件分支、持久化、人机协同
- 🟡 [[llama-index]] — LlamaIndex数据框架，连接LLM与私有数据，支持RAG、Agent、工作流
- 🟡 [[openai-agents]] — OpenAI官方Agents SDK，支持Responses API、工具调用、网络搜索、文件搜索
- 🟡 [[phidata]] — Phidata构建AI助手的框架，支持知识库、结构化输出、多模态(已合并为Agno)
- 🟡 [[praisonai]] — PraisonAI低代码多Agent框架，基于CrewAI，支持AutoGen集成和UI界面
- 🟡 [[pydantic-ai]] — PydanticAI类型安全Agent框架，Samuel Colvin(Pydantic作者)出品，强类型验证
- 🟡 [[semantic-kernel]] — Microsoft Semantic Kernel SDK，连接LLM、记忆、规划器和插件
- 🔴 [[smolagents]] — HuggingFace SmolAgents极简Agent库，代码优先，最小抽象，高度透明
