# 容器编排/K8s机器学习运维

> Kubernetes机器学习运维工具(KServe、Kubeflow)

## 命令列表

- 🔴 [[kfctl apply]] — Deploy Kubeflow to Kubernetes cluster
- 🔴 [[kfctl delete]] — Delete Kubeflow deployment
- 🟡 [[kfp experiment create]] — Create a Kubeflow Pipelines experiment
- 🟡 [[kfp pipeline list]] — List Kubeflow Pipelines
- 🟡 [[kfp pipeline upload]] — Upload a pipeline to Kubeflow Pipelines
- 🟡 [[kfp run submit]] — Submit a Kubeflow Pipeline run
- 🟡 [[kubectl apply -f inferenceservice.yaml]] — Deploy a KServe InferenceService
- 🔴 [[kubectl delete inferenceservice]] — Delete a KServe InferenceService
- 🟡 [[kubectl describe inferenceservice]] — Show detailed information about KServe InferenceService
- 🟡 [[kubectl get experiments.kubeflow.org]] — List Kubeflow Katib experiments
- 🟡 [[kubectl get inferenceservices]] — List KServe InferenceServices
- 🟡 [[kubectl get notebooks]] — List Kubeflow Notebooks
- 🟡 [[kubectl get pytorchjobs]] — List Kubeflow PyTorch training jobs
- 🟡 [[kubectl get tfjobs]] — List Kubeflow TensorFlow training jobs
- 🟡 [[kubectl get workflows]] — List Kubeflow Pipelines workflows
- 🟡 [[kubectl logs -l serving.kserve.io/inferenceservice]] — View logs for KServe model serving pods
- 🟡 [[kubectl port-forward svc/istio-ingressgateway]] — Access Kubeflow dashboard locally
- 🟡 [[kubectl port-forward svc/ml-pipeline-ui]] — Access Kubeflow Pipelines UI locally
