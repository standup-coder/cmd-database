# AI基础设施/Harness工程

> LLM评测Harness、Agent评测、安全测试与基准测试工程化工具

## 命令列表

- 🔴 [[agent-bench]] — AgentBench多场景Agent能力评测，涵盖OS、数据库、知识图谱、数字卡牌、网页浏览
- 🟡 [[api-bench]] — APIBench API理解评测，测试LLM理解REST API文档并生成正确调用的能力
- 🟡 [[arena]] — LMSYS Chatbot Arena众测平台，人类盲测对比不同LLM的对话质量
- 🔴 [[big-bench]] — BIG-bench (Beyond the Imitation Game) 大规模行为评测，200+语言理解任务
- 🔴 [[ds-1000]] — DS-1000数据科学代码评测基准，1000个真实数据科学问题，测试pandas/numpy/sklearn等库使用
- 🔴 [[humaneval]] — HumanEval代码生成评测基准，164个Python编程问题，测试函数级代码生成能力
- 🔴 [[humaneval-exec]] — HumanEval代码评测执行器，运行模型生成的代码并通过单元测试，计算pass@k指标
- 🟡 [[judge-eval]] — LLM-as-Judge评估框架，系统化评估裁判模型质量，减少位置偏见和长度偏见
- 🔴 [[livecodebench]] — LiveCodeBench实时代码评测，持续更新的编程题 benchmark，防数据污染，反映模型真实代码能力
- 🟡 [[llm-as-judge]] — LLM-as-Judge方法论实现，位置偏见校正、一致性检验、多维度评分
- 🟡 [[long-context-eval]] — 长上下文评测工具，Needle-in-Haystack、RULER等，测试模型在超长文本中的信息检索能力
- 🟡 [[math-eval]] — 数学能力评测工具，支持GSM8K、MATH、SVAMP等数据集，验证模型推理和计算准确性
- 🔴 [[mbpp]] — MBPP (Mostly Basic Python Programming) 代码生成评测，974个Python编程任务
- 🟡 [[mt-bench]] — MT-Bench多轮对话评测基准，80个高质量多轮问题，GPT-4作为裁判评分
- 🟡 [[multilingual-eval]] — 多语言能力评测，支持C-Eval(中文)、MMLU多语言、XCOPA、XNLI等跨语言基准
- 🟡 [[pair-wise-eval]] — Pair-wise成对比较评测，ELO评分系统，适用于模型A/B测试和排序
- 🟡 [[prometheus-eval]] — Prometheus开源LLM评估模型，无需GPT-4即可进行高质量指令评测
- 🔴 [[red-teaming]] — PyRIT (Python Risk Identification Toolkit) 微软开源红队测试框架，自动化LLM安全测试
- 🟡 [[safety-eval]] — MLSafety模型安全评测套件，覆盖毒性、偏见、隐私泄露、越狱抗性
- 🟡 [[swe-bench]] — SWE-bench软件工程评测基准，测试LLM解决真实GitHub Issue的能力
- 🟡 [[tool-bench]] — ToolBench工具使用评测基准，16000+ API，测试LLM的工具选择、调用、组合能力
