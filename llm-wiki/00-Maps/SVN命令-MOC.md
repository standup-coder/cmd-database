# 版本控制/SVN命令

> Apache Subversion版本控制系统命令

## 命令列表

- 🟡 [[svn add]] — Add files to version control
- 🟡 [[svn checkout]] — Check out a working copy from a repository
- 🟡 [[svn cleanup]] — Clean up working copy (remove locks, fix interrupted operations)
- 🟡 [[svn commit]] — Send changes to repository
- 🟡 [[svn copy]] — Create a copy (branch/tag) in repository
- 🔴 [[svn delete]] — Remove files from version control
- 🟡 [[svn diff]] — Show differences between revisions
- 🟡 [[svn info]] — Display information about working copy or repository
- 🟡 [[svn log]] — Show commit log messages
- 🔴 [[svn merge]] — Merge changes from another branch
- 🔴 [[svn revert]] — Undo local changes
- 🟡 [[svn status]] — Show working copy status
- 🟡 [[svn switch]] — Switch working copy to different URL
- 🟡 [[svn update]] — Update working copy to latest revision
