# 容器编排/K8s故障排查

> Kubernetes故障排查和诊断专用工具及命令集

## 命令列表

- 🟡 [[kubectl api-resources]] — 列出所有可用的 API 资源
- 🟡 [[kubectl api-versions]] — 显示支持的 API 版本
- 🟡 [[kubectl auth can-i]] — 检查用户权限
- 🟡 [[kubectl cluster-info]] — 显示集群基本信息
- 🟡 [[kubectl config current-context]] — 显示当前配置上下文
- 🟡 [[kubectl debug]] — 创建调试容器进行故障排查
- 🟡 [[kubectl describe pod]] — 详细查看 Pod 状态和事件信息
- 🟡 [[kubectl exec -it]] — 进入容器执行网络诊断命令
- 🟡 [[kubectl get componentstatuses]] — 检查 Kubernetes 控制平面组件健康状态
- 🟡 [[kubectl get events]] — 获取集群事件信息，用于故障诊断
- 🟡 [[kubectl get limitrange]] — 检查资源限制范围
- 🟡 [[kubectl get resourcequota]] — 检查资源配额限制
- 🟡 [[kubectl logs]] — 查看容器日志用于问题诊断
- 🟡 [[kubectl port-forward]] — 端口转发用于本地调试和测试
- 🟡 [[kubectl top]] — 查看资源使用情况，识别性能瓶颈
