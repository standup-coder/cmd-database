# AI基础设施/模型架构

> 大模型架构分析、配置管理、注意力可视化与参数量计算，深入理解Transformer及变体

## 命令列表

- 🔴 [[accelerate-config]] — HuggingFace Accelerate配置向导，交互式生成分布式训练配置文件
- 🟡 [[bertviz]] — BertViz Transformer注意力可视化工具，交互式查看各层各头的注意力权重分布
- 🟡 [[calflops]] — CalFlops模型参数量与FLOPs计算工具，精确统计前向/反向传播计算量
- 🟡 [[diffusers-cli]] — HuggingFace Diffusers扩散模型工具，管理文生图/视频模型管线、调度器配置
- 🟡 [[einops]] — Einops张量操作语义化库，清晰表达reshape、transpose、reduce等维度变换操作
- 🟡 [[mamba]] — Mamba状态空间模型(SSM)实现，线性复杂度序列建模，替代Transformer注意力机制
- 🟡 [[modelcards]] — HuggingFace Model Card工具，生成和管理模型文档，确保模型可追溯和合规
- 🟡 [[timm]] — PyTorch Image Models库，800+预训练视觉模型，涵盖ViT、ConvNeXt、Swin Transformer等
- 🟡 [[transformers-cli]] — HuggingFace Transformers命令行工具，查看模型配置、下载模型、转换格式
