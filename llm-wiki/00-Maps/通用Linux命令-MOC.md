- 🟡 [[awk]] — 文本处理语言，按模式扫描和处理文件
- 🟡 [[cat]] — 连接文件并输出到标准输出
- 🟡 [[cd]] — 切换当前工作目录
- 🟡 [[cp]] — 复制文件或目录
- 🔴 [[crontab]] — 管理用户定时任务
- 🟡 [[df]] — 报告文件系统磁盘空间使用情况
- 🟡 [[du]] — 估算文件和目录的磁盘使用空间
- 🟡 [[egrep]] — 扩展正则表达式grep（等同于grep -E）

- 🟡 [[free]] — 显示系统内存使用情况
- 🟡 [[grep]] — 在文件中搜索文本模式
- 🟡 [[gzip]] — GNU 文件压缩工具
- 🟡 [[head]] — 显示文件开头部分
- 🟡 [[htop]] — 交互式进程查看器
- 🟡 [[jq]] — 命令行 JSON 处理器
- 🟡 [[kill]] — 向进程发送信号以终止或控制

- 🟡 [[pwd]] — 显示当前工作目录的完整路径

- 🔴 [[rm]] — 删除文件或目录
- 🟡 [[rmdir]] — 删除空目录
- 🟡 [[rsync]] — 远程和本地文件同步工具，支持增量传输
- 🟡 [[scp]] — 基于 SSH 的安全文件复制
- 🟡 [[sed]] — 流编辑器，执行基本的文本转换
- 🟡 [[sh]] — Bourne Shell，标准的Unix命令解释器
- 🟡 [[ssh]] — OpenSSH 远程登录客户端
- 🔴 [[sudo]] — 以超级用户或其他用户身份执行命令
- 🟡 [[tail]] — 显示文件末尾部分，支持实时跟踪
- 🟡 [[timedatectl]] — 查看和设置系统时间与日期
- 🟡 [[tmux]] — 终端复用器，支持会话持久化和多窗口
- 🟡 [[trash]] — 将文件移动到回收站而非永久删除
- 🟡 [[uptime]] — 显示系统运行时长和平均负载
- 🟡 [[whereis]] — 定位命令的二进制、源码和手册页文件
- 🟡 [[which]] — 查找命令的可执行文件路径
- 🔴 [[xargs]] — 从标准输入构建并执行命令
- 🔴 [[yq]] — 命令行 YAML/JSON/TOML 处理器
- 🟡 [[zsh]] — Z Shell，功能强大的交互式Shell，兼容Bash
