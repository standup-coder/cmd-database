# 容器编排/K8s集群管理

> Kubernetes集群管理与控制平面工具

## 命令列表

- 🟡 [[etcdctl get]] — Get value of specified key from etcd
- 🔴 [[etcdctl member list]] — List all etcd cluster members
- 🔴 [[etcdctl snapshot restore]] — Restore etcd cluster from backup snapshot
- 🟡 [[etcdctl snapshot save]] — Create backup snapshot of etcd data
- 🟡 [[journalctl -u kubelet]] — View kubelet service logs
- 🟡 [[kubeadm config view]] — View the kubeadm configuration
- 🔴 [[kubeadm init]] — Initialize Kubernetes control plane node
- 🔴 [[kubeadm join]] — Join worker node to Kubernetes cluster
- 🔴 [[kubeadm reset]] — Reset node state and undo kubeadm init or join
- 🔴 [[kubeadm token list]] — List bootstrap tokens on the cluster
- 🔴 [[kubeadm upgrade]] — Upgrade Kubernetes cluster to specified version
- 🟡 [[kubectl label nodes]] — Add or remove labels from nodes
- 🟡 [[kubectl taint nodes]] — Add or remove taints from nodes
- 🟡 [[systemctl status kubelet]] — Check kubelet service status
