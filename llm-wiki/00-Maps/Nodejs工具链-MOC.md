# 编程语言/Node.js工具链

> Node.js运行时与npm包管理工具

## 命令列表

- 🟡 [[node]] — Execute JavaScript code using Node.js runtime
- 🟡 [[npm audit]] — Check for security vulnerabilities
- 🟡 [[npm install]] — Install package dependencies
- 🟡 [[npm outdated]] — Check for outdated packages
- 🔴 [[npm publish]] — Publish package to npm registry
- 🟡 [[npm run]] — Run arbitrary package scripts
- 🟡 [[npm start]] — Run the start script defined in package.json
- 🟡 [[npm test]] — Run the test script defined in package.json
- 🟡 [[npm update]] — Update packages to their latest versions
- 🟡 [[npx]] — Execute npm package binaries
