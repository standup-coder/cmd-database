# npm start

> Run the start script defined in package.json

## 安装

```bash
Included with Node.js
```

## 用法

```
npm start
```

## 示例

### 示例 1: Start the application

```bash
npm start
```

### 示例 2: 启动并传递自定义端口参数

```bash
npm start -- --port=3000
```
