# 容器编排/Kubernetes命令

> Kubernetes命令行容器编排管理工具

## 命令列表

- 🟡 [[kubectl]] — Kubernetes命令行工具，管理集群资源
- 🟡 [[kubectl-annotate]] — Update annotations on a resource
- 🔴 [[kubectl-apply]] — Apply a configuration to a resource by filename or stdin
- 🔴 [[kubectl-config]] — Modify kubeconfig files
- 🟡 [[kubectl-cordon]] — Mark node as unschedulable
- 🟡 [[kubectl-create]] — Create a resource from a file or stdin
- 🔴 [[kubectl-delete]] — Delete resources by filenames, resources and names, or by label selectors
- 🟡 [[kubectl-describe]] — Show detailed information about a resource
- 🔴 [[kubectl-drain]] — Drain node in preparation for maintenance
- 🔴 [[kubectl-exec]] — Execute a command in a container
- 🟡 [[kubectl-get]] — Display one or many resources
- 🟡 [[kubectl-label]] — Update labels on a resource
- 🟡 [[kubectl-logs]] — Print the logs for a container in a pod
- 🟡 [[kubectl-port-forward]] — Forward one or more local ports to a pod
- 🔴 [[kubectl-rollout]] — Manage the rollout of a resource
- 🔴 [[kubectl-scale]] — Set a new size for a deployment, replica set, or replication controller
- 🟡 [[kubectl-top]] — Display resource (CPU/memory) usage
