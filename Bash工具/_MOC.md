- 🟡 [[bash]] — Bash Shell 解释器，执行脚本文件或交互式命令

- 🟡 [[shfmt]] — Shell 脚本格式化工具
- 🟡 [[source]] — 在当前 Shell 环境中执行脚本（不创建子进程）
