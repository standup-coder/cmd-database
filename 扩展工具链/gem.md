# gem

> Ruby 包管理器

## 安装

```bash
随 Ruby 安装
```

## 用法

```
gem [OPTIONS] COMMAND
```

## 参数

| Flag | Description |
|------|-------------|
| `install` | 安装 |
| `list` | 列出 |
| `uninstall` | 卸载 |
| `update` |  |

## 示例

### 示例 1: 安装 Rails

```bash
gem install rails
```

### 示例 2: 列出已安装 gem

```bash
gem list
```
