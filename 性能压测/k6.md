# k6

> 现代化负载测试工具，使用 JavaScript 编写测试脚本

## 安装

```bash
参考 https://grafana.com/docs/k6/latest/set-up/install-k6/
```

## 用法

```
k6 [OPTIONS] [COMMAND]
```

## 参数

| Flag | Description |
|------|-------------|
| `run` | 运行测试脚本 |
| `--vus` | 虚拟用户数 |
| `--duration` | 测试持续时间 |

## 示例

### 示例 1: 使用 10 个虚拟用户运行 30 秒测试

```bash
k6 run --vus 10 --duration 30s script.js
```

### 示例 2: 在 k6 Cloud 上运行分布式测试

```bash
k6 cloud run script.js
```
