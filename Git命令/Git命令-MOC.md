# 版本控制/Git命令

> Git分布式版本控制系统命令

## 命令列表

- 🟡 [[git-add]] — 添加文件到暂存区
- 🟡 [[git-branch]] — 列出、创建或删除分支
- 🟡 [[git-checkout]] — 切换分支或恢复文件
- 🟡 [[git-clone]] — 克隆远程仓库到本地
- 🟡 [[git-commit]] — 提交暂存区的更改
- 🟡 [[git-diff]] — 显示工作区与暂存区或提交之间的差异
- 🟡 [[git-fetch]] — 从远程仓库下载对象和引用，但不合并
- 🟡 [[git-init]] — 初始化新的Git仓库
- 🟡 [[git-log]] — 查看提交历史
- 🟡 [[git-merge]] — 合并分支
- 🟡 [[git-pull]] — 拉取远程更改并合并
- 🔴 [[git-push]] — 推送本地提交到远程仓库
- 🟡 [[git-rebase]] — 将当前分支变基到另一分支，重写提交历史
- 🟡 [[git-restore]] — 恢复工作区或暂存区文件（Git 2.23+ 推荐替代checkout）
- 🟡 [[git-show]] — 显示提交、标签或对象的详细信息和差异
- 🟡 [[git-status]] — 显示工作区状态
- 🟡 [[git-switch]] — 切换分支（Git 2.23+ 推荐替代checkout）
