# uptime

> 显示系统运行时长和平均负载

## 用法

```
uptime
```

## 示例

### 示例 1: 查看系统已运行时间和负载

```bash
uptime
```

### 示例 2: 以人类可读格式显示运行时长

```bash
uptime -p
```
